# 終焉ノ黙示録 ― Eclipse Requiem ―

これはp5.jsで制作されたストーリー型RPGです。

---

## 🎮 ゲーム概要
黒い月が現れた世界で、主人公が“終焉因子”に目覚め、
運命と戦うダークファンタジーゲームです。

---

## 🕹 操作方法
- ENTER：決定・進行
- キーボード入力：名前入力

---

## 📖 遊び方
- 名前を入力して物語を開始
- ストーリーを読み進める
- 今後、選択肢や戦闘が追加されます

---

## ⚔️ 特徴
- 名前によって物語が変化
- ダークファンタジー×厨二病ストーリー
- 今後、分岐・戦闘・仲間システム追加予定
